
>>> Install XAMPP
    https://www.apachefriends.org/download.html

>>> Install Composer
    https://getcomposer.org/download/

>>> Install NodeJS
    https://nodejs.org/en/download

>>> Clone the project in the GitHub repository
    https://github.com/JosephAndreyDacera/Peercademy.git

1. Start your XAMPP server
2. Open your browser and search for http://localhost/phpmyadmin/
3. Create a new database and name it "peer_academy"
4. Open the porject on VS Code
5. Setup .env file
    If .env file don't exist open vs code ternimal and execute the command below
        cp .env.example .env
        php artisan key:generate
    Then setup the following in the .env file
        APP_NAME=Peercademy
        DB_CONNECTION=mysql
        DB_HOST=127.0.0.1
        DB_PORT=3306
        DB_DATABASE=peer_academy
        DB_USERNAME=root
        DB_PASSWORD=
